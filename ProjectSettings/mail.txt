В данном приложении для отправки уведомлений администратору используется почтовый сервис mail.ru

Для настройки почты на сервисе mail.ru нужно создать почтовый ящик и указать сторонее приложение для отправки

со сторонних приложений.
 
Создайте в настройках аккаунта на сайте Mail.ru специальный пароль, 

уникальный для каждого стороннего приложения, с помощью которого вы хотите читать Почту.
 
1.	Войдите в аккаунт ваша_почта@mail.ru через Mail.ru на компьютере.
 
2.	Перейдите в настройки Mail ID → «Безопасность» → «Пароли для внешних приложений».
 
3.	Нажмите «Добавить». (Требуется указать название и нужен номер вашего сотового телефона).

4.  Дальнейшие изменения нужно внести в файл inc\mailConfig.php.
 
5.	Скопируйте новый пароль и сохраните его в значение переменой MAIL_PASSWORD.

6.  Название почтового ящика сохраните в значение переменой MAIL_USER_NAME.

7.  Укажите сервер исходящей почты в случае почтового сервиса mail.ru  это 'smtp.mail.ru' в значение переменной SMTP_SERVER.

8.  Укажите порт в случае почтового сервиса mail.ru это 465 в значение переменной MAIL_PORT.

9.  Укажите протокол шифрования в случае почтового сервиса это 'SSL' в значение переменной ENCRYPTION_TYPE.

10.  Включите отправку почты для этого нужно изменить значение переменной IS_SEND_MAIL установкой в значение true.
